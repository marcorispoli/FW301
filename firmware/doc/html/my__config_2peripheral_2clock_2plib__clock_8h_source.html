81/* Function:
82 void CLOCK_Initialize (void);
83
84 Summary:
85 Initializes all the modules related to the system clock.
86
87 Description:
88 This function initializes the clock as defined by the MHC and Clock Manager
89 selections. The function will configure the NVM Flash Wait states based on
90 the configured CPU operational frequency. It will then configure the
91 oscillators.
92
93 For each of the clock sources (External Oscillator, Digital Phase Locked
94 Loop, Internal 48MHz Oscillator, External 32KHz oscillator and the Internal
95 32KHz oscillator) enabled in MHC, the function will configure the clock
96 settings and will then wait till the clock is ready. In case of DPLL, the
97 function will wait till a lock is obtained.
98
99 The function will then configure the Generic clock generators based on MHC
100 configurations. If a Generic Clock is enabled in MHC, this will be enabled
101 in the CLOCK_Initialize() function. The function will apply the CPU clock
102 divider and will wait for the Main Clock module to get ready. If the Main
103 Clock to the Peripheral APB and AHB interfaces was enabled in MHC, these
104 will be enabled in the CLOCK_Initialize() function. If the Peripheral Clock
105 Channels were enabled in MHC, these will be enabled in the
106 CLOCK_Initialize() function.
107
108 The peripheral AHB and APB main clock and peripheral channel clocks will be
109 enabled when the peripheral specific initialize functions are called. This
110 will override the setting in MHC. The Generic Clock Generator source for
111 desired peripheral channel must be configured in MHC.
112
113 Precondition:
114 MHC GUI should be configured with the right values. Incorrect configuration
115 of the Clock will result in incorrect peripheral behavior or a non
116 functional device.
117
118 Parameters:
119 None.
120
121 Returns:
122 None.
123
124 Example:
125 <code>
126 CLOCK_Initialize();
127 </code>
128
129 Remarks:
130 This function should be called before calling other Clock library functions.
131*/
132
133void CLOCK_Initialize (void);
