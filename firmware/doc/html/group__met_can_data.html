◆ MET_CAN_PROTOCOL_ERROR_DEFS

enum MET_CAN_PROTOCOL_ERROR_DEFS

Can Protocol Error codes.

Those errors are generated into the module in case of some function should fail. When a error is detected, the error is provided to an internal error handler: MET_CanOpen_Error_Callback()
