Detailed Description

Rx and Tx communication data.

This structure is internally used to handle the communication data frames. The structure is local and is not shared with the application. The Received data are stroeed into the rx_message[] array as soon as the data is received into the FIFO. The tx_message[] array is passed to the Can Sender function when a frame shall be sent.
